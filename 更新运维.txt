RAI v0.10.9.62 public release notes

Release date: 2026-07-02

This public file is intentionally sanitized for GitHub.

Security and privacy policy for this repository:
- Do not commit API keys, passwords, JWT secrets, TOTP secrets, provider tokens, or service account credentials.
- Do not commit SQLite databases, database WAL/SHM files, uploads, avatars, runtime reports, or user-generated content.
- Do not commit VPS IPs, SSH ports, server paths, production backup paths, database hashes, account emails, or other private operations data.
- Keep production environment variables and full deployment records in private operations storage only.

本次更新重点

- 管理员后台新增“限制风控”能力：可配置单用户并发请求数、每分钟额度、5 小时额度、每周额度，并可配置上传限频、单用户上传总容量、文件数量、PWA 任务奖励和邀请奖励结算策略。
- 修复上传可被账号批量滥用打满磁盘的风险：/api/upload 现在有独立频率限制、单文件大小限制、单用户总容量限制、单用户文件数限制；默认免费用户上传上限为 100MB / 50 files，删除用户时会同步清理用户文件记录与本地文件。
- 加固积分和会员权益风控：新用户欢迎点数为 200；PWA 任务奖励增加后台开关和账号年龄门槛；邀请奖励默认进入待结算，不再注册即自动结算。
- 管理员登录状态修复：管理员 token 校验会刷新有效凭据，避免隔天打开后台时因为旧 token 残留导致只能用无痕模式重新登录。
- 管理员后台新增“邮件群发”入口：支持先发测试邮件，再确认发送给所有已验证邮箱用户；后端增加主题/正文长度限制、测试邮箱校验和显式群发确认，降低误发风险。
- 改善多设备同步：对话同步轮询和窗口恢复刷新更及时，新对话标题会更快更新到侧边栏。
- 修复小屏 iPhone 和 iOS 浏览器输入框聚焦后页面被自动放大的布局问题，移动端输入控件统一保持不低于 16px。
- 修复研究模式中用户插话的位置：插话现在进入子 agent 讨论上下文，不再插入到主对话 AI 气泡上方。
- 修复搜索来源链接点击无反应的问题，并对来源 URL 做安全规范化。
- 修复聊天区顶部/底部渐变遮罩遮挡文字的问题：滚动到最顶部或最底部时自动隐藏遮罩。
- 优化输入框 + 号菜单和模型菜单：展开/收起补齐动画过渡，全部模型菜单固定 x 轴，不再左右漂移。
- 新增浏览器标签页标题动画设置：支持静态 RAI、问候语动画、对话标题、用户自定义文字和跑马灯模式；问候语跟随当前语言。
- 当 AI 回复完成且用户没有查看当前标签页时，浏览器会发出提醒，方便回到对话。
- RAI 思考内容改为正文位置内的浅色小字号区域，回复完成后自动折叠成无描边圆角按钮，可点击展开/收起。
- 侧边栏历史对话默认按时间分组：今天显示具体时间，较早对话显示月日或年月；更新过的旧对话会自然移动到上方。
- AI 询问用户的交互改成圆角无描边卡片：支持多个问题、左右切换、选项点击自动进入下一题，自定义回答兼容中文输入法并通过确认按钮提交。

Public changelog since 0.10.9.21
